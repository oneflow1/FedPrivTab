# FedPrivTab Vue 前端展示遗漏审查

审查依据：`docs/requirements.md`、历史前端原型行为、当前 `frontend/src/App.vue`，并在远端 `http://110.40.166.66:8501/` 用 Playwright 抽样验证。

## 结论

当前 Vue 前端已具备登录、导航、角色权限、客户端账号列表、CSV 上传、后端预处理、训练调用、报告生成等主流程入口，但相较需求文档和历史原型展示，仍存在较多展示层遗漏。主要问题不是接口不可用，而是很多展示只是标题或占位文案，没有实际图表、表格或配置控件。

## 高优先级遗漏

1. 数据分析页图表未实现
   - 需求：字段分布图、单特征分布、特征均值、相关性热力图。
   - 历史原型：使用 Plotly histogram/bar/heatmap 实际渲染。
   - 当前 Vue：仅显示标题和下拉框，未渲染任何图形或均值/相关性数据。
   - 证据：`frontend/src/App.vue:209`、`frontend/src/App.vue:210`。

2. 数据分析页指标不完整
   - 需求：样本数、字段数、数值特征数、缺失值数量。
   - 当前 Vue：只展示样本数和字段数。
   - 证据：`frontend/src/App.vue:207`。

3. 实验训练页参数控件缺失
   - 需求：客户端数量、通信轮数、集中式 Epoch、隐藏层数量、隐藏层神经元、激活函数、学习率，以及 Batch Size、本地 Epoch、客户端采样比例、Dirichlet alpha、聚合方式、裁剪阈值、噪声倍率、epsilon、delta。
   - 历史原型：这些都有 number_input/selectbox/slider。
   - 当前 Vue：仅展示客户端数量、通信轮数、集中式 Epoch、隐藏层神经元数，缺隐藏层数量、激活函数、学习率和全部高级联邦/DP 参数控件。
   - 证据：`frontend/src/App.vue:226`。

4. 实验训练页数据版本选择未实现
   - 需求：支持选择数据版本。
   - 历史原型：集中式版本和联邦版本分别 selectbox，并限制来源。
   - 当前 Vue：只显示“集中式 MLP 数据版本”“FedAvg / DP-FedAvg 数据版本”两行文字，没有 select 控件，也没有版本来源约束。
   - 证据：`frontend/src/App.vue:228`。

5. 结果分析页展示严重不足
   - 需求：Accuracy、Precision、Recall、F1、AUC；训练曲线摘要、混淆矩阵、客户端分布、差分隐私参数。
   - 历史原型：三方案对比表、指标对比柱状图、隐私-性能对比、按方案 tabs、混淆矩阵、客户端分布、DP 参数、Markdown 预览/源码/下载。
   - 当前 Vue：只展示 Accuracy/Precision/Recall/F1 表格和生成报告按钮；缺 AUC、训练曲线、混淆矩阵、客户端分布、DP 参数、报告下载和预览模式。
   - 证据：`frontend/src/App.vue:232`、`frontend/src/App.vue:233`。

## 中优先级遗漏

1. 数据预处理默认推荐策略未按需求实现
   - 需求：缺失率高推荐删除；数值/数字字符串推荐中位数；类别推荐众数。
   - 当前 Vue：所有列默认 `mode`，没有根据列类型/缺失率推荐；标准化默认对所有非 target/client_id 列 standard，包括类别列。
   - 证据：`frontend/src/App.vue:119`、`frontend/src/App.vue:120`。

2. 数据预处理“缺失值摘要”信息不足
   - 需求：上传后刷新目标变量、缺失值摘要、数值标准化和数据预览。
   - 当前 Vue：有每列处理方式选择，但不显示缺失数量、缺失率、推荐策略。
   - 证据：`frontend/src/App.vue:218`。

3. 数值候选字段识别不准确
   - 需求：数值或数字字符串字段应识别为数值型候选字段。
   - 当前 Vue：标准化候选只排除 `target` 和 `client_id`，不做实际数值识别，类别列也会被纳入。
   - 运行证据：上传样例后 `category` 出现在“数值标准化”选择中。

4. 统计摘要不是完整 `describe(include='all')`
   - 需求：展示完整统计摘要。
   - 历史原型：`frame.describe(include="all").transpose()`。
   - 当前 Vue：仅字段、类型、非空数、缺失值、均值。
   - 证据：`frontend/src/App.vue:48`、`frontend/src/App.vue:208`。

5. CSV 解析较简化
   - 当前 `csvPreview` 直接按逗号 split，不能正确处理引号、逗号转义、换行字段。
   - 证据：`frontend/src/App.vue:37`。

## 已满足/基本满足

1. 登录页可用，错误提示存在；远端浏览器已验证管理员登录成功。
2. 首页展示客户端数量、训练结果数量和三类方案，且没有冗余“数据状态”。
3. 客户端管理页仅管理员/研究员可见，客户端用户导航不显示该页和实验训练页。
4. 客户端管理页固定显示 client-1 至 client-4，并提供改密入口；未出现新增/删除/修改我的密码入口。
5. 数据分析页与数据预处理页分离，上传状态能保留在页面切换期间。
6. 数据预处理调用 `/preprocess` 使用 multipart FormData，不是按行 JSON 上传。
7. 远端服务、API 和基础登录流程可用。

## 建议修复顺序

1. 先补训练页参数和数据版本选择，否则完整实验流程不符合需求。
2. 再补结果分析页的 AUC、曲线、混淆矩阵、客户端分布、DP 参数和报告下载。
3. 然后补数据分析页真实图表和完整统计摘要。
4. 最后修 CSV 解析、推荐策略、数值字段识别和展示细节。
